# Limitations

MarketImmune V1 is a benchmark and safety demo, not an exchange-grade surveillance
system.

- Real data is public Binance USD-M Futures market data only.
- Agent behavior, labels, harm events, and scenario manifests are synthetic.
- V1 cannot identify real private accounts.
- V1 cannot prove market manipulation by a real person or institution.
- V1 does not trade.
- V1 does not forecast prices for profit.
- V1 does not cover cross-CEX or on-chain behavior.
- V1 benchmark numbers are valid only when generated by the repository scripts.
